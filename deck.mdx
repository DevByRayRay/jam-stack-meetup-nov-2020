# Gatsby
## Wat is het?

- Gratis, open source framework gebouwd op React
- Om websites en applicatie's mee te bouwen die super snel laden
- Werkt met GraphQL
- Heeft plugins voor o.a. Shopify, Stripe, WordPress en nog veel meer

---

### 3.1 Wat heb je nodig?

<Steps>

1. Git
2. NodeJS

</Steps>

---

### 3.2 Installatie

`$ npm install gatsby-cli -g`

---

### 3.3 Test installatie

`$ gatsby --help`

---

### 3.4 Nieuw Gatsby project

`$ gatsby new hello-world gatsby new hello-world https://github.com/gatsbyjs/gatsby-starter-hello-world`

---

### 3.5 Start project

`$ cd hello-world`

`$ gatsby develop`

---


### 3.6 Open browser

Open [http://localhost:8000](http://localhost:8000)
