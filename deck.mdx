# Gatsby
## Hoe werkt het?

1. Gatsby heeft een content bron nodig: Markdown files, JSON, CMS, Headless CMS etc.
2. Gatsby maakt op basis van GraphQL een Data Layer om de data tijdens het bouw process op te halen.
3. Tijdens het bouwen word een GraphQL server opgezet voor alle opgehaalde data.
4. Na het ophalen smelt Gatsby de content en React componenten samen tot een website.
5. De gebruiker kan daardoor supersnel de website op zijn scherm krijgen.

# Netlify Functions
## Wat is het?

Serverless functions zijn JavaScript bestanden waarin je dingen kunt berekenen of externe API's kunt aanroepen. 

Zonder dat je hiervoor een backend server hoeft te maken.

---

# Netlify Functions
## Hoe ziet het er uit?

Het kan zo simpel zijn als dit:

```
exports.handler = async (event, context) => {
  return {
   statusCode: 200,
   body: JSON.stringify({
    message: 'Hello world"
   }),
  }
}
```

---

# Netlify Functions
## Wat is het resultaat?

Dit is dan het response van de function:

`{"message": "Hello world"}`
