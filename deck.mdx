## Wat is Gatsby?

<Steps>

1. Gratis, open source framework gebouwd op React
2. Om websites en applicatie's mee te bouwen die super snel laden
3. Haald zijn content uit Markdown of andere bronnen zoals Shopify, WordPress en nog veel meer
4. Heeft plugins voor o.a. Shopify, Stripe, WordPress, Paypal, en nog veel meer

</Steps>

---

## 1. Waar gebruik je Gatsby voor?

<Steps>
1. Als je het CMS en website wilt loskoppelen van elkaar
2. Wanneer de website en CMS los van elkaar moeten kunnen functioneren.
</Steps>

### 3.2 Installatie

### 3.3 Test installatie

`$ gatsby --help`

---

### 3.4 Nieuw Gatsby project

`$ gatsby new hello-world gatsby new hello-world https://github.com/gatsbyjs/gatsby-starter-hello-world`

---

### 3.5 Start project

`$ cd hello-world`

`$ gatsby develop`
